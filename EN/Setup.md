- # Setup Guide (Uploading to VRChat)

  ## 1. Preparation

  - Download and install Unity (2022.3.22f1)
  - Install VRChat Creator Companion (VCC)

  ![VCC0](/Manual_Pict/VCC0.png)

  ---

  ## 2. Required Packages

  Install the following packages via VCC:

  - Required Packages
    - lilToon → Add Repository from website
    - FaceEmo → Download (VCC)
    - Modular Avatar → Download from website

  ---

  ## 3. Create a New Project

  Create a new project from "Create New Project".

  ![VCC1](/Manual_Pict/VCC1.png)

  Select "Unity 2022 Avatar Project", set a project name, and create it.

  ![VCC2](/Manual_Pict/VCC2.png)

  ---

  ## 4. Install Packages

  Open "Manage Project".

  ![VCC3](/Manual_Pict/VCC3.png)

  Install:
  - lilToon
  - Modular Avatar
  - FaceEmo

  If not visible, repeat Step 2.

  ![VCC7](/Manual_Pict/VCC7.png)

  ---

  ## 5. Import Horoyoel

  - Open the project

  ![OpenProject](/Manual_Pict/OpenProject.png)

  - Assets → Import Package → Custom Package

  ![Unity1](/Manual_Pict/Unity1.png)

  - Import `HoroyoelVerXXX.unitypackage`

  ![Unity2](/Manual_Pict/Unity2.png)

  ---

  ## 6. Place in Scene

  Drag & drop:

  - Horoyoel_PresetA (Angel outfit)
  - Horoyoel_PresetR (Roomwear)

  ![Unity3](/Manual_Pict/Unity3.png)

  Success:

  ![Unity4](/Manual_Pict/Unity4.png)

  ---

  ## 7. Upload

  - VRChat SDK → Show Control Panel

  ![SDK1](/Manual_Pict/SDK1.png)

  - Login and configure

  ⚠️ Visibility must be **Private**  
  Public upload is a violation of the terms.

  ![SDK2](/Manual_Pict/SDK2.png)

  - Build & Publish

  Done!
